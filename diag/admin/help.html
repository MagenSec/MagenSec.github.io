Dashboard Help & Explanations

This page provides details on how the metrics and visualizations in the Command Center dashboard are calculated.

Key Performance Indicators (KPIs)

Security Score

The Security Score is a high-level metric (0-100) representing your organization's overall security posture. A higher score is better.

It is calculated using a weighted formula based on:

  • Vulnerability Density: The ratio of vulnerable applications to the total number of applications.
  • Average Exploitability: The average ExploitProbability across all discovered applications.

The goal is to provide an at-a-glance understanding of your risk exposure.

Managed Devices

This is the total count of unique devices (endpoints) that have reported telemetry data.

Unique Applications

This is the total count of distinct application names found across all managed devices.

High-Risk Assets

This counts the number of devices that have at least one application with a critical vulnerability (ExploitProbability > 0.7).

Charts & Visualizations

Security Posture (Last 30 Days)

This chart shows the trend of your Security Score over the past 30 days.

Note: Currently, this chart uses a simulated projection for demonstration purposes. In a production environment, this would be populated with historical snapshot data of the daily security score.

Vulnerability Breakdown

This donut chart categorizes all vulnerable applications based on their ExploitProbability into three buckets:

  • Critical: Probability > 70%
  • High: Probability between 40% and 70%
  • Medium: Probability <= 40%

Top 5 Riskiest Applications

This bar chart highlights the applications with the highest ExploitProbability, allowing you to prioritize remediation efforts on the most significant threats.
