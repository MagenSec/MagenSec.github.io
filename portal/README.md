# MagenSec Portal

> Modern web portal for device security management and organization administration

**Live Portal**: [https://magensec.github.io/portal/](https://magensec.github.io/portal/)

## Features

- **Organization Management**: Team-based device monitoring
- **Secure Authentication**: OAuth-based login
- **Real-Time Updates**: Live data refresh
- **Modern UI**: Responsive design for desktop and mobile
