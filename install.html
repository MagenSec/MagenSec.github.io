Getting Started

Install in 3 Steps

1

Install from Microsoft Store

Open the Microsoft Store listing and install MagenSec. Windows selects the right package automatically. If you need silent rollout or license-key activation, use the manual and CLI workflow below instead.

2

Create or Sign In to Your Account

Open the MagenSec portal at magensec.gigabits.co.in/portal and create a free account or sign in. Your device registers automatically on first launch.

3

MAGI Starts Protecting Immediately

Within minutes of installation, MagenSec scans your device, identifies vulnerabilities, and begins real-time protection. Your first security report appears in the portal dashboard.
